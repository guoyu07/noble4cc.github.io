C++笔记-typedef关键字

Veröffentlicht am 2014-12-03   |   in C++   |  

typedef是c/c++的关键字，用途是给类型定义别名。类似于#define，但与define有很大的不同。

typedef用法总结

typedef主要分为两种种用途一个是给变量一个易记且意义明确的新名字，另一个是简化一些比较复杂的类型声明。

给变量定义别名

比如：
typedef char CHAR
char c;CHAR c;作用是一样的！
主要常见的是在C语言定义结构体中简化变量定义！

还有就是定义平台无关的变量
比如：
在一种计算机上支持的类型很多
long num;是可以定义的，但是换一种计算机不支持long类型，你就必须将所所有的long转换成int或者其他支持的类型。可移植性很差！但如果这样写：
typedef long LONG;
LONG num;
在其他不支持long类型的计算机上就可以将tepedef long LONG 改写成typedef int LONG；
不必将所有的long进行替换就可以实现移植！c++中大量使用了这种方法！比如说 size_t 类型，就是通过typedef实现的！

简化声明

对复杂变量建立一个类型别名的方法很简单，你只要在传统的变量声明表达式里用类型名替代变量名，然后把关键字typedef加在该语句的开头就行了
比如：int (a[5])(int,char);是不是很复杂，这是个数组指针，指向一个存储返回值为int，参数为int和char的函数指针的长度为5的数组
typedef int (pFun)(int, char*);
pFun a[5];等价于上面麻烦的定义

示例代码 

#include <iostream>
typedef char* CHAR;
typedef int (*pType)(int,char*);
int func(int i,char* c)
{
    std::cout<<i<<std::endl;
    std::cout<<*c<<std::endl;
}
int func2(int i,char* c)
{
    std::cout<<*c<<std::endl;
    std::cout<<i<<std::endl;
}
int main()
{
    CHAR c;//定义char类型的指针
    char word='c';
    c=&word;
    pType a[2];
    a[0]=func;//存储函数1
    a[1]=func2;//存储函数2
    a[0](1,c);//执行函数
    a[1](2,c);
    return 0;
}

PS:typedef用于简化声明在c++中很常见，但初期学习会比较吃力，必须仔细阅读代码，动手操作，很快就可以掌握！
